User story 1: Как пользователь, при открытии сайта я хочу видеть баннер с рендомно-сгенерированный системой проект c его кратким описанием, для того, чтобы заинтересовать меня.
Сценарий: Я захожу на сайт и вижу баннер на весь экран с заглавной картинкой проекта, на котором отображены краткое описание проекта и кнопка для перехода в данный проект.

User story 2: Как пользователь, при открытии сайта я хочу видеть сверху над баннером возможность выполнить поиск интересующего меня проекта, по ключевым словам, для того что бы меньше тратить времени на поиск проекта.
Сценарий: В строке поиска ввожу текст (наименование проекта и ключевые слова) и по нажатию на кнопку получаю список сформированных, согласно моему запросу, проектов.

User story 3: Как пользователь, при открытии сайта я хочу видеть сверху над баннером возможность просмотра правил пользования услугами сайта, для того что понять насколько подходит данный ресурс для создания проекта.
Сценарий: Я захожу на сайт и вижу сверху над баннером поле «About us» and «Rules» и по нажатию на кнопку открывается окно с описанием правилами использования данного сайта (как создавать проекты, как жертвовать деньги и т.д.).

User story 4: Как пользователь, при открытии сайта я хочу видеть снизу под баннером с проектом иконки для открытия страниц kickstarter на facebook, vkontakte, Instagram, Pinterest, YouTube и Twitter, для того что бы всегда вовремя просматривать новый и свои проекты.
Сценарий: Я захожу на сайт и вижу под баннером с проектом в левом нижнем углу брендированные иконки соответствующих ресурсов для открытия kickstarter на facebook, vkontakte, Instagram, Pinterest, YouTube и Twitter.

User story 5: Как пользователь, при открытии сайта я хочу видеть в правом верхнем углу поля для регистрации авторизации, для того что бы в дальнейшем меньше уменьшить трату времени на заполнение личных данных.
Сценарий: Я захожу на сайт и вижу в правом верхнем углу поля «Sign up» и «Log in» и при нажатии на соответствующую кнопку открывается окно для регистрации или авторизации пользователя.

User story 6: Как пользователь, при открытии поля регистрации, была возможность синхронизации с facebook или vkontakte, для упрощения процесса регистрации на сайте.
Сценарий: В открытом окне для регистрации или авторизации пользователя вижу возможность авторизироваться или залогиниться через социальные сети.

User story 7: Как пользователь, при открытии сайта я хочу видеть сверху над баннером возможность быстрого создания нового проекта, для того что бы упростить процесс создания нового проекта.
Сценарий: Я захожу на сайт и вижу сверху над баннером с проектом кнопку для создания нового проекта. При нажатии на кнопку открывается новое окно для заполнения информации о новом проекте.

User story 8: Как пользователь, при открытии сайта я хочу видеть возможность выбора проектов по странам, чтобы ускорить процесс нахождения проектов «земляков».
Сценарий: Я захожу на сайт и вижу под баннером возможность выбора просмотра проектов по странам. При выборе необходимой страны, выводит на экран список всех проектов граждан выбранной страны.
